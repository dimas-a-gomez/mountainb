# MountainB | Source Code

Bienvenido al núcleo de **MountainB**. Esta carpeta contiene el código fuente, la lógica del framework y los estilos que dan vida al tema.

### 🛠 Stack de Desarrollo
Este proyecto está construido sobre el ecosistema **Hamlet**, utilizando:
* **HBS (Handlebars):** Para la estructura de plantillas.
* **SCSS:** Para la arquitectura de estilos modular.
* **Node.js:** Para la automatización y compilación (`npm`).

### 📦 Estructura del Proyecto
- `/customization`: Variables globales y configuraciones de diseño.
- `/elements`: Componentes reutilizables del tema.
- `/styles`: Hojas de estilo organizadas bajo metodología modular.

### 🚀 ¿Cómo compilar?
Si has realizado cambios en el código y deseas generar el XML final para Blogger:

1. Asegúrate de tener las dependencias instaladas:
   `npm install`
2. Ejecuta el proceso de construcción:
   `npm run build`

El archivo resultante se generará en la carpeta `/dist`.
