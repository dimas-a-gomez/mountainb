# Contribuyendo a MountainB

Gracias por tu interés en mejorar MountainB. Valoramos enormemente cualquier tipo de feedback que nos ayude a perfeccionar este framework.

---

## Cómo colaborar

Aunque MountainB es un proyecto de código abierto donde eres libre de modificar y comercializar el tema, apreciamos mucho cuando la comunidad decide compartir sus mejoras:

1. **Reportar Bugs:** Si encuentras un error, por favor abre un *Issue* en GitHub incluyendo:

* Una descripción clara del problema.
* Pasos para reproducirlo.
* Capturas de pantalla si es posible.

2. **Sugerencias y Mejoras:** ¿Tienes una idea para una nueva funcionalidad o una optimización? ¡Nos encantaría escucharla! Abre un *Issue* con la etiqueta `enhancement`.

3. **Código (Pull Requests):** Si has escrito código que mejora el proyecto, siéntete libre de enviar un *Pull Request*. Lo revisaremos con gusto.

---

## Ética de Comunidad

* Sé respetuoso con otros desarrolladores.
* No hay presión para contribuir; el software es libre. Sin embargo, tu feedback es el motor que hace que Engines siga creando herramientas de alta calidad.

---
