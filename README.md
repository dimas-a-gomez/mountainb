# MountainB

MountainB es un framework de diseño profesional para Blogger, optimizado para velocidad, minimalismo y estética SaaS. Creado por **MicroTecN Engines**.

---

## 📥 Para Usuarios (Instalación Rápida)

Si solo deseas utilizar el tema en tu blog sin realizar modificaciones al código:

1. Navega a la carpeta [`dist/`](https://github.com/dimas-a-gomez/mountainb/tree/main/dist).
2. Descarga el archivo [`mountainb.xml`](https://github.com/dimas-a-gomez/mountainb/blob/main/dist/mountainb.xml).
3. En tu panel de Blogger, ve a **Tema > Restaurar** y sube este archivo.
